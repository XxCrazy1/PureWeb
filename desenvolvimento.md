Desenvolver uma extensão moderna para o navegador Opera (Chromium-based) focada em bloqueio inteligente de anúncios e elementos intrusivos, priorizando performance, privacidade e compatibilidade com os padrões atuais da Web.

A extensão deve utilizar APIs oficiais do Manifest V3 (como declarativeNetRequest e injeção de CSS/JS leve) para:

Filtrar anúncios de forma eficiente e não intrusiva

Remover banners, pop-ups, overlays e trackers visuais

Minimizar impactos no carregamento e na experiência do usuário

Operar de forma discreta, mantendo o comportamento do site o mais próximo possível do original

Requisitos técnicos:

Código limpo, modular e bem documentado

Arquitetura escalável para futuras listas de filtros

Interface simples (popup) com toggle on/off e status de bloqueio

Foco em privacidade do usuário (nenhuma coleta de dados)

Alta compatibilidade com sites modernos (SPA, React, Vue, etc.)

O objetivo é criar uma extensão robusta, profissional e sustentável, alinhada às boas práticas de desenvolvimento de extensões para navegadores Chromium.
